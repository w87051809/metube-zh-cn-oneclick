# v1.1.1

## 本次发布

这是一个界面文字修订版本。

## 变化

- 网页左上角品牌名从 `MeTube` 改成 `视频下载`。
- 浏览器标签页标题改成 `视频下载`。
- `YouTube 登录入口` 移到页面右上角，改成顶部工具条，不再占表单下面一整块位置。
- 安装脚本缓存版本号更新到 `20260720-4`，避免浏览器继续加载旧前端脚本。

## 验证

- `node --check metube-zh-cn.js`
- Python AST 语法检查：`ytdl.py`、`subscriptions.py`、`dl_formats.py`
- `git diff --check`
- 敏感信息扫描
- GitHub Actions 质量门禁

## 升级说明

已安装用户刷新页面即可看到变化。浏览器如果仍显示旧文字，按 `Ctrl+F5` 强刷新。

## 回滚说明

如果要恢复原显示，把 `metube-zh-cn.js` 中 `MeTube` 的翻译恢复为 `MeTube`，并把入口页标题恢复为 `MeTube`。
