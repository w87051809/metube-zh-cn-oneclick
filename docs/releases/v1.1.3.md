# v1.1.3

## 本次发布

这是一次订阅列表展示增强版本。

## 变化

- 订阅列表的名称前面增加频道头像小图标。
- 小图标尺寸固定为 24 x 24，不改变原来的表格行高。
- 后端订阅记录新增展示字段：
  - `thumbnail`
  - `channel_id`
  - `channel_url`
  - `uploader_id`
- 新增订阅会立刻保存频道头像信息。
- 旧订阅会在下一次订阅检查后自动补齐频道头像信息。
- 安装脚本缓存版本号更新到 `20260720-6`。

## 验证

- `node --check metube-zh-cn.js`
- Python AST 语法检查：`subscriptions.py`、`ytdl.py`、`dl_formats.py`
- `git diff --check`
- 敏感信息扫描
- 服务器 `bash -n install.sh`
- 服务器 MeTube 服务健康检查
- GitHub Actions 质量门禁

## 升级说明

已安装用户刷新页面即可看到订阅列表头像。旧订阅如果先显示文字占位，点一次“检查”或等下一次自动订阅检查后会补成频道头像。

## 回滚说明

如果要回滚，只需要恢复上一版 `metube-zh-cn.js`、`subscriptions.py` 和入口页脚本版本号。服务器下载目录和视频文件不受影响。
