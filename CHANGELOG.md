# 变更日志

本文件记录公开版本的重要变化。

以后不再为每个很小的文档改动单独发 Release。正式 Release 只用于用户可感知功能、安装脚本、修复、安全和发布流程变化。

## [Unreleased]

- 暂无。

## [1.1.0] - 2026-07-20

### Added

- 增加正式发布流程文档：`docs/release-process.md`
- 增加 GitHub Actions 质量门禁：语法检查、敏感信息扫描、本地地址扫描。
- 增加 Pull Request 模板，要求写清变更、验证和安全检查。
- 增加 GitHub Release Notes 分类配置。
- 增加本版本的标准 Release Notes 文件：`docs/releases/v1.1.0.md`

### Changed

- README 增加“发布与质量”说明，明确以后按变更日志和发布流程发版。

### Security

- 发布前检查明确禁止把服务器地址、密码、Cookie、密钥、安装日记写进公开仓库。

## [1.0.13] - 2026-07-20

### Changed

- 删除 README 顶部临时作者和贡献者块。

## [1.0.12] - 2026-07-20

### Fixed

- `检查状态` 按钮点击后显示明确反馈，避免看起来没有反应。

## [1.0.9] - 2026-07-20

### Changed

- 网站右下角 `GitHub` 链接指向本仓库。

## [1.0.7] - 2026-07-20

### Fixed

- 新订阅会补下 24 小时内新发布的视频，避免刚发布的视频被当成老视频跳过。
- 下载临时文件放到视频目录里的 `.下载临时文件`。

## [1.0.6] - 2026-07-20

### Added

- 增加 `YouTube 登录入口`，可上传 `cookies.txt`。

## [1.0.5] - 2026-07-20

### Added

- 下载前尽量把英文标题翻译成中文，并保存到网页记录和硬盘文件名。

### Changed

- 视频格式交给 `yt-dlp` 自动选择，提高成功率。

## [1.0.1] - 2026-07-19

### Added

- 增加素材包下载：视频、SRT 字幕、JPG 封面可一起添加。

## [1.0.0] - 2026-07-18

### Added

- 首次发布 MeTube 中文一键安装版。
